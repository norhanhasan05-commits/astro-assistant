# Astro Assistant

Render-ready mobile web application.

## Render settings

- Language: Python
- Build Command: `pip install -r requirements.txt`
- Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
- Root Directory: leave empty

## Features

- Natal positions
- Topocentric houses
- Transits
- Solar Return
- Secondary Progressions
- Aspect tables
- Experimental Primary Directions for ASC/MC with Naibod key

The astrology interpretation is educational and is not scientifically validated.
